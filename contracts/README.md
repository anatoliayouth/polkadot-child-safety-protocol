# Smart Contracts

This directory will contain the ink! smart contracts for the Polkadot Child Safety Protocol.

## Planned Contracts

- `child-safety-contract`: Main contract for managing guardian policies and child safety rules
- `did-registry-contract`: Integration with KILT DIDs for identity management
- `blocklist-contract`: Contract for managing blocked entities and content filters

## Development

Contracts will be written in ink! and deployed to Polkadot parachains.
