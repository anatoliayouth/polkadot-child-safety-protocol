# Documentation

This directory contains comprehensive documentation for the Polkadot Child Safety Protocol.

## Documentation Structure

- `api/`: API documentation and integration guides
- `architecture/`: System architecture and design decisions
- `tutorials/`: Step-by-step tutorials for different user types
- `security/`: Security considerations and best practices
- `research/`: Research papers and background information
